# Shoriful Islam Shaon — Portfolio Website

Neon Cyberpunk style personal portfolio website.

## 🚀 How to Host on GitHub Pages

1. Create a new repo on GitHub (e.g. `shaon-portfolio`)
2. Upload all 3 files: `index.html`, `style.css`, `script.js`
3. Go to repo **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)`
5. Click **Save** — your site will be live at:
   `https://YOUR-USERNAME.github.io/shaon-portfolio`

## 📁 Files
- `index.html` — Main HTML structure
- `style.css`  — All styles & animations
- `script.js`  — Particles, typing, scroll reveal, count-up

## ✨ Features
- Neon cyberpunk design
- Animated particle background
- Glitch hover effect on name
- Golden gradient subtitle
- Scroll reveal animations
- Count-up stats
- Mobile responsive
- Active nav highlight
